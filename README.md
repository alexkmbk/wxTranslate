# wxTranslate

![alt tag](https://github.com/alexkmbk/wxTranslate/blob/master/resources/Screenshot.png)

**wxTranslate** is a lightweight desktop application for Windows that provides instant text translation using a simple and intuitive interface.  
Features include:
- Global hotkeys for quick translation of selected text
- System tray integration
- Automatic language detection
- Support for multiple favorite languages
- Fast and convenient copy/paste operations

**Global Hotkeys:**
- <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>T</kbd> — Show the main window
- <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>E</kbd> — Translate currently selected text

Built with wxWidgets and C++20.
